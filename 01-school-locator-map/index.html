Project Overview

This project aims to create a comprehensive geographic database of schools within the Kumasi Metropolitan Assembly, providing valuable insights for educational planning, resource allocation, and community development.

Data Collection

Systematic gathering of school locations, types, and facilities information across Kumasi.

Geospatial Analysis

Mapping and analyzing school distribution patterns using QGIS for informed decision-making.

Data Visualization

Creating interactive dashboards in Power BI to communicate insights effectively.

Project Objectives

  • Create a comprehensive database of all schools within the Kumasi Metropolitan Assembly
  • Analyze spatial distribution patterns of educational facilities
  • Identify areas with insufficient educational infrastructure
  • Provide data-driven insights for future educational planning
  • Develop interactive visualization tools for policymakers
